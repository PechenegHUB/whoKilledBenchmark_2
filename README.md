# README — запуск и работа с решением

Проект представляет собой приложение для анализа микрофотографий полированных шлифов. Решение определяет технологический класс образца, оценивает долю талька, строит маски и карты уверенности, а также позволяет запускать как одиночную, так и пакетную обработку изображений.

---

## Запуск проекта

Перейдите в папку проекта и установите зависимости:

```bash
pip install -r requirements.txt
```

После этого запустите приложение:

```bash
streamlit run app.py
```

Если в проекте используются готовые скрипты запуска, можно запустить их:

```bash
run_app.bat
```

для Windows или:

```bash
./run_app.sh
```

для Linux/macOS.

После запуска откройте приложение в браузере:

```text
http://localhost:8501
```

---

## Основные режимы работы

В приложении есть несколько рабочих разделов:

1. **Анализ одного шлифа**  
   Используется для детальной проверки одного изображения. Пользователь загружает снимок, выбирает нужные расчёты и получает итоговый класс, долю талька, маску, карты уверенности и визуальные наложения.

2. **Пакетная обработка**  
   Используется для обработки сразу нескольких изображений или ZIP-архива. После запуска приложение формирует таблицу результатов по всем файлам, сохраняет маски, визуализации и отчёты.

3. **Экспертная проверка и доразметка**  
   Позволяет открыть сохранённый результат, проверить предсказание модели, скорректировать итоговый класс, добавить комментарий и вручную исправить маску в браузерном редакторе.

4. **Журнал и экспорт**  
   Показывает сохранённые результаты запусков и позволяет выгружать таблицы, отчёты, изображения, маски и архивы.

---

## Выбор расчётов перед запуском

Перед анализом можно выбрать, какие артефакты считать:

- базовую классификацию образца;
- долю талька;
- итоговую маску;
- подробную карту `talc-score`;
- Grad-CAM;
- карту общей уверенности;
- superpixel-сегментацию;
- сравнение разных масок;
- overlay-визуализации;
- файлы для отчётов и выгрузки.

Базовые метрики и итоговая маска считаются всегда. Дополнительные карты можно отключить, чтобы ускорить обработку, особенно в пакетном режиме.

## Одиночный анализ

Для анализа одного изображения:

1. Откройте раздел **«Анализ одного шлифа»**.
2. Загрузите изображение шлифа.
3. Выберите нужные расчёты.
4. Нажмите кнопку запуска анализа.
5. Посмотрите итоговый класс, долю талька, маску и карты уверенности.
6. При необходимости перейдите к экспертной проверке и исправьте маску вручную.

Одиночный режим удобен для детального разбора спорных образцов и проверки качества предсказания.

---

## Пакетная обработка

Для массовой обработки:

1. Откройте раздел **«Пакетная обработка»**.
2. Загрузите несколько изображений или ZIP-архив.
3. При необходимости заполните метаданные партии.
4. Выберите нужные расчёты.
5. Запустите обработку.
6. После завершения выгрузите таблицу, маски, overlay-изображения, отчёты или архив с результатами.

Пакетный режим нужен для быстрой обработки серии шлифов без ручного запуска каждого изображения отдельно.

---

## Экспертное редактирование

В разделе экспертной проверки можно:

- открыть уже обработанный шлиф;
- посмотреть исходное изображение и маску модели;
- изменить итоговый класс;
- добавить комментарий эксперта;
- вручную исправить маску;
- использовать масштабирование;
- пользоваться ластиком;
- сохранить экспертную оценку;
- сохранить исправленную маску для дальнейшего использования.

Экспертные правки помогают уточнять спорные области и накапливать более качественные данные для дальнейшего улучшения решения.

## Результаты работы

Результаты сохраняются в папку:

```text
outputs/
```

Внутри могут сохраняться:

```text
outputs/images/              # входные изображения
outputs/masks/               # маски талька
outputs/class_masks/         # итоговые классовые маски
outputs/confidence_maps/     # карты уверенности
outputs/talc_scores/         # talc-score карты
outputs/gradcams/            # Grad-CAM карты
outputs/overlays/            # изображения с наложением маски
outputs/expert_masks/        # экспертные исправленные маски
outputs/reports/             # PDF-отчёты
outputs/reviews/             # CSV/JSON-журнал результатов
```

Основной журнал результатов:

```text
outputs/reviews/analysis_results.csv
```

## Рекомендуемые сценарии

**Быстрый прогон партии:**  
оставить только базовую классификацию, долю талька и итоговую маску.

**Подробный экспертный анализ:**  
включить `talc-score`, Grad-CAM, карту уверенности, superpixel-сегментацию и сравнение масок.

**Финальная выгрузка:**  
после обработки сформировать таблицу, отчёты, overlay-изображения и ZIP-архив результатов.

## Если что-то не работает

Если приложение не открылось, проверьте, что команда запуска выполняется из корневой папки проекта:

```bash
streamlit run app.py
```

Если браузер не открылся автоматически, перейдите вручную:

```text
http://localhost:8501
```

Если результаты не обновляются после изменения файлов, очистите кэш Streamlit:

```bash
streamlit cache clear
```

и перезапустите приложение.
